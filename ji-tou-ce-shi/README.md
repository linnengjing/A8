# 机头测试

完整及部分测试机头编织系统的状态。

**触摸**：单、双击机头测试字符区，进入机头测试界面。  
**按键**：方向键移动光标到指定位置，按 **Enter** 键进入机头测试界面。

本页面主要测试：

机头编织部分控制三角\(左右接针、吊目、翻针\)、选针器、度目电机、生克电机、换梭电磁铁、编织换向传感器等项目。
