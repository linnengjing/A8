# 可变度目编辑



![](https://raw.githubusercontent.com/HQwangyun/HQ-image/master/%E5%8F%AF%E5%8F%98%E5%BA%A6%E7%9B%AE%E7%BC%96%E8%BE%91.png)

◆触摸：触笔选择可变度目编辑字符区，进入可变度目编辑页

| 功能键 | 说明 |
| :--- | :--- |
| PAT-CNT | 跳至CNT文件编辑 |
| 定位 | 跳转至指定行/列 |
| 开始 | 跳转至花型开始界面 |
| 结束 | 跳转至花型结束界面 |
| 左翻 | 向左翻列 |
| 右翻 | 向右翻列 |
| 上翻 | 向上翻行 |
| 下翻 | 向下翻行 |
| 修改 | 修改当前光标停留处的色码 |
| 放大 | 放大当前显示界面 |
| 缩小 | 修小当前显示界面 |
