# 可变度目编辑

**触摸**：触笔选择**\[可变度目编辑\]**图标，点击进入可变度目编辑页，根据编织需要设置相应参数，修改完成后点击右上角\[×\]退出当前操作。

| 功能键 | 说明 |
| :---: | :---: |
| PAT-CNT | 跳至CNT文件编辑 |
| 定位 | 跳转至指定行/列 |
| 开始 | 跳转至花型开始界面 |
| 结束 | 跳转至花型结束界面 |
| 左翻 | 向左翻列 |
| 右翻 | 向右翻列 |
| 上翻 | 向上翻行 |
| 下翻 | 向下翻行 |
| 修改 | 修改当前光标停留处的色码 |
| 放大 | 放大当前显示界面 |
| 缩小 | 修小当前显示界面 |
